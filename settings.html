Zwei-Faktor-Authentifizierung

Schütze dein Konto zusätzlich mit einem Einmal-Code aus einer Authenticator-App (Google Authenticator, SelfAuthenticator …).

1. App öffnen → Konto hinzufügen → QR-Code scannen (oder Schlüssel manuell eintippen).
2. Den angezeigten 6-stelligen Code unten eingeben und bestätigen.

2FA ist aktiv. Zum Deaktivieren bitte dein Passwort eingeben.
